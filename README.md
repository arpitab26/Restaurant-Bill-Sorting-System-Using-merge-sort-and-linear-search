🍽️ Restaurant Bill Sorting System
🚀 Using Merge Sort & Linear Search in C
📌 Project Overview

This project is a menu-driven Restaurant Bill Management System built in C programming that demonstrates the practical application of Data Structures & Algorithms.

It allows users to:

Store multiple restaurant bills
Sort bills efficiently based on amount
Search for a specific bill instantly using Bill ID

👉 The system combines Merge Sort (efficient sorting) and Linear Search (simple searching) to simulate a real-world billing scenario.

🎯 Objective

To design and implement a system that:

Organizes billing records efficiently
Demonstrates algorithmic problem solving
Applies theoretical concepts to real-world use cases
⚙️ Features

✨ User-Friendly Menu System

Display all bills
Sort bills
Search bills
Exit program

📊 Sorting

Uses Merge Sort (Divide & Conquer)
Sorts bills in ascending order of amount

🔍 Searching

Uses Linear Search
Finds bill using Bill ID

💾 Data Handling

Stores:
Bill ID
Customer Name
Amount
🧠 Algorithms Used
🔹 Merge Sort
Divides bill list into smaller parts
Recursively sorts them
Merges into a sorted list

⚡ Time Complexity: O(n log n)

🔹 Linear Search
Checks each bill sequentially
Stops when match is found

⚡ Time Complexity: O(n)

🏗️ System Workflow
Enter number of bills
Input bill details
Choose operation from menu
Perform:
Display
Sort
